4. FILE PERMISSIONS & OWNERSHIP

Understanding Permissions

View permissions


ls -l                                                           # Long format shows permissions
ls -la                                                          # Include hidden files
stat file.txt                                                   # Detailed file status including permissions

Permission format: -rwxrwxrwx


Position 1: file type (- file, d directory, l link)
Positions 2-4: owner (user) permissions
Positions 5-7: group permissions
Positions 8-10: others permissions
r=read(4), w=write(2), x=execute(1)

Change Permissions (chmod)

Symbolic mode


chmod u+x file                                                  # Add execute for user
chmod g-w file                                                  # Remove write for group
chmod o+r file                                                  # Add read for others
chmod a+x file                                                  # Add execute for all
chmod u+rw,g+r file                                             # Multiple changes
chmod ugo+x file                                                # Execute for user, group, others

Numeric (octal) mode


chmod 644 file                                                  # rw-r--r-- (common for files)
chmod 755 file                                                  # rwxr-xr-x (common for scripts)
chmod 777 file                                                  # rwxrwxrwx (DANGEROUS - avoid)
chmod 600 file                                                  # rw------- (private file)
chmod 700 directory                                             # rwx------ (private directory)

Recursive


chmod -R 755 directory                                          # Apply to directory and contents
chmod -R u+x directory                                          # Add execute recursively

Special permissions


chmod u+s file                                                  # Set SUID (setuid)
chmod g+s directory                                             # Set SGID (setgid)
chmod +t directory                                              # Set sticky bit
chmod 4755 file                                                 # SUID + rwxr-xr-x
chmod 2755 directory                                            # SGID + rwxr-xr-x
chmod 1777 directory                                            # Sticky + rwxrwxrwx (like /tmp)

Change Ownership (chown)

Change owner


chown newowner file                                             # Change file owner
chown -R newowner dir                                           # Recursive

Change owner and group


chown user:group file                                           # Change both
chown user: file                                                # Change user, group to user's primary group

Change only group (alternative to chgrp)


chown :group file                                               # Change group only

Change Group (chgrp)


chgrp newgroup file                                             # Change group
chgrp -R newgroup dir                                           # Recursive

Access Control Lists (ACL)

View ACLs


getfacl file.txt                                                # Show ACL for file
getfacl -R directory                                            # Recursive

Set ACLs


setfacl -m u:username:rwx file                                  # Modify user ACL
setfacl -m g:groupname:rx file                                  # Modify group ACL
setfacl -m o::r file                                            # Modify others ACL
setfacl -x u:username file                                      # Remove user ACL
setfacl -b file                                                 # Remove all ACLs
setfacl -R -m u:username:rx dir                                 # Recursive
setfacl -d -m u:username:rwx dir                                # Set default ACL for new files

Copy ACLs


getfacl file1 | setfacl --set-file=- file2                      # Copy ACL from file1 to file2

File Attributes (Extended)

View attributes

Change attributes
