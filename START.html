🎮 How Each Feature Works

🔍 Search (Ctrl+Shift+F)

  1. Press Ctrl+Shift+F
  2. Type what you're looking for (e.g., "docker")
  3. Press Enter or click "Search"
  4. Results show below - matching text highlighted
  5. Click "Reset" to show all commands again

📋 Copy Commands

  1. Find the command you want
  2. Hover over it - a "Copy" button appears
  3. Click "Copy"
  4. See "Copied!" message
  5. Paste anywhere with Ctrl+V

🌙 Dark Mode (Ctrl+Shift+T)

  1. Press Ctrl+Shift+T
  2. Page switches to dark theme instantly
  3. Or click the 🌙 button (top-right)
  4. Your preference is saved
  5. Dark mode will be ON next time you visit

📥 Download as HTML

  1. Go to any Part page (1, 2, or 3)
  2. Look for purple "Download" section at top
  3. Click "Download as HTML"
  4. File saves to your computer
  5. You can open it offline anytime

🖨️ Print / Save as PDF

  1. Press Ctrl+P or click "Print" button
  2. Print dialog opens
  3. Choose "Save as PDF" (if you want PDF)
  4. Click "Save"
  5. File saves to your computer

🐛 Troubleshooting

Problem: HTML files won't open

Solution: Right-click → "Open with" → Choose browser (Chrome, Firefox, Edge, Safari)

Problem: "Python not found" error

Solution: You don't need Python to use the app! Just double-click .html files instead. Or download Python from python.org

Problem: Dark mode not saving

Solution: Enable cookies in browser settings

Problem: Copy button not working

Solution: Open in a modern browser (Chrome, Firefox, Edge). Older browsers may not support this.
