🔌 API Endpoints (Flask Backend)

Available When Flask Server Running

Start with: python app.py

GET /api/commands

Description: Get all commands with optional filter

Usage: /api/commands?filter=docker

Returns: JSON list of matching commands

{
  "total": 5,
  "commands": [
    {
      "title": "List containers",
      "code": "docker ps",
      "file": "devops-tools.html"
    }
  ]
}

GET /api/statistics

Description: Get page statistics

Usage: /api/statistics

{
  "total_commands": 100,
  "total_parts": 3,
  "last_updated": "2024-01-07T...",
  "files": ["system-admin.html", "devops-tools.html", "monitoring-security.html"]
}

POST /api/search

Description: Advanced search with filters

Usage: POST body with JSON

{
  "query": "kubernetes",
  "file": "devops-tools.html"
}

GET /api/export

Description: Export commands in different formats

Usage: /api/export?format=json or ?format=csv

Formats: json, csv

🎨 Customization Options

Easy Customizations:

  • Colors: Edit :root variables in styles.css
  • Fonts: Change font-family in styles.css
  • Port: Change port number in app.py
  • Shortcuts: Modify in scripts.js
  • Commands: Add to HTML files directly

Examples:

Change primary color from blue to green:

/* In styles.css */
--primary-color: #27ae60;  /* Change from #2c3e50 */

Change Flask port from 5000 to 8080:

# In app.py
app.run(debug=True, host='0.0.0.0', port=8080)
