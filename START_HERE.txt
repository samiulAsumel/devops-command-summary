🐛 TROUBLESHOOTING

Problem: "Files won't open in browser"
→ Right-click file → Open with → Choose Chrome/Firefox/Safari/Edge

Problem: "Python not found" (if using Flask)
→ Install Python from https://www.python.org/
→ Make sure to check "Add Python to PATH"

Problem: "Styles/scripts not loading"
→ Clear browser cache: Ctrl+Shift+Delete
→ Make sure all files are in same folder

Problem: "Port 5000 already in use"
→ Edit app.py, change port=5000 to port=8080

For more help, see README.md or QUICKSTART.md
